Types of Tags
  • Container Tag :

Tag that containce many other tags under it are known as container tag. HTML is the biggest container tag.
  • Ambeded Tag :

Tag that are used under container tags are called Ambeded tags. They don't contain any other tag under it.
  • Open Tag :

Tag without closing tags are known as open tag. hr and br tags are without closing tag
  • Closing Tag :

Tag with closing tags are known as paired tag.
Basic tags of HTML
  • html Tag :

Html tag tells the web browser that this is an html document. It is printing tag of html and hole program is written under this tag. It is also called root element.
  • head Tag :

Head tag defines the header porsion of the web page. It contains mata data(data about data).
  • title Tag :

It defines the title of the web page which is visible as top of the page.
  • body Tag :

It describe the visible page content and it is the main tag of html where contents are written.
  • font Tag :

This tag is used to apply different font styles, colors and size on html content.
  • heading Tag :

Heading tag is used to add a heading on the web page and heading1 (h1) is the largest heading and heading6 (h6) is the smallest heading of the html.
  • marquee Tag :

This tag is used to add scrolling text or image on a web page.
  • a Tag :

Anchor tag is used to create a hyper link with html element. href (hyper text refrence) attribute is used to add address of destination file.
  • form Tag :

A form is an area that contain multiple alignment like text, radio, buttons, checkbox, dropdown list etc. It has following elements.
  • input Tag :

It is the most used form element to accept input from user.
  • select Tag :

This tag is used to create a dropdown list. Option tag are used within select tag to define the value of selection.
  • textarea Tag :

This tag is used to accept multiple set of characters words, paragraph etc. It is mainly used to get address from user.
  • frameset Tag :

This tag is used to define number of frames of a web page. The frameset element holds multiple frame element to add a separate html document or an image.
  • img Tag :

This tag is used to add an image on html web page.
  • table Tag :

This tag is used to define table on the html page.
  • tbody Tag :

It represents the main content of a table, encompassing the data rows.
  • th Tag :

This tag is used to create a table headings(th).
  • tr Tag :

This tag is used to create table row(tr).
  • td Tag :

This tag is used to create table data(td).
  • center Tag :

This tag is used to align the content at the center of the web page.
  • ol Tag :

It is used to define the ordered list(OL).
  • ul Tag :

It is used to define th unordered list(UL).
  • li Tag :

It is used to define the content of list within the ordered or unordered list.
  • dl Tag :

It is used to define a description list or definition list(DL).
  • dt Tag :

It is used to define the definition term(DT) or list items.
  • dd Tag :

It contains the actual content about definition term. It is called as definition data(DD).
  • pre Tag :

This tag is used to accept spaces and new line characters from the editor screen. It should be always used within body tag and it is not a close tag.
  • br Tag :

This tag is used to break the line. It breaks the line without starting a new paragraph and BR tag is a not close tag.
  • hr Tag :

This tag is used to create a horigentle line on the web page and HR is also not a close tag.
Formatting tags of HTML
  • p Tag : This tag is used to create a paragraph on the web page.
  • u Tag : To make the text underline.
  • b Tag : To make the text bold.
  • strong Tag : It also make the text bold.
  • i Tag : To make the text italic.
  • em Tag : It also make the text emphasized/italic.
  • s Tag : To make the text strikethrough.
  • del Tag : It also make the text strikethrough.
  • big Tag : It makes the text bigger as compare to normal text.
  • small Tag : It makes the text smaller as compare to normal text.
  • sub Tag : To write the content at the base position.
  • sup Tag : To write the content at the top position.
  • mark Tag : To make the text highlighted.
  • tt Tag : To write the content in the web page as typewriter writing.
  • q Tag :To add quotesion on a html content.
