Attributes of Basic Tags
Attributes of body tag
Attibutes Name Value Description
bgcolor colour name to define the background color of the web page
background imgage address to define the background image of the web page
text colour name to define the text colour on the web page
Attributes of font tag
color colour name to define the color of the text
face font name to apply desired font style on html an element
size numeric value(1-7) to define the size of html content
Attributes of hr tag
color colour name to define the color of horigentle line
width px / % to define the width of horigentle line
size numeric value to define the size of horigentle line
Attributes of ol and ul tag
type (ol): A, a, I, i, 1
(ul): square, circle, disc		 to define the type of bullet
start number to define the starting value of numbered list
Attributes of frameset tag
rows % to define the number or rows
cols % to define the number or columns
Attributes of table tag
border numeric to define the thickness of border
bgcolor colour name to define the background-colour of the table
height px / % to define the height of the table
width px / % to define the width of the table
align left / right / center to define the alignment of the table
cellspacing numeric to define the space between cell content and cell border
cellpadding numeric to define the space between border of the cell
colspan number of columns to merge multiple columns in to single column
rowspan number of rows to merge multiple rows in to single row
Attributes of marquee tag
behavior scroll, swipe, alternate, slide to define the behavior of scrolling text
scrollamount numeric to define the speed of scrolling text
scrolldelay numeric it adds the delay in scrolling text
bgcolor colour name to define the background-colour of marquee tag content
width px / % to define the width of marquee tag
height px / % to define the height of marquee tag
direction left / right / up / down to set direction of scrolling text
vspace numeric it provides a verticle space around marquee tag element
hspace numeric it provides a horigentle space around marquee tag element
Attributes of img tag
src (source) image address to define the address of an image
alt (alternate) alternate name of an image to define an alternate value if image is not visible
width px / % to define the width of an image
height px / % to define the height of an image
align left / right / center to define the alignment of an image on web page
border numeric to define the border of an image
Attributes of input tag
type text to accept text information from user in the form of alphabet, number, symbol etc
radio to create a radio button in which user has limited choise
checkbox to accept more than one choise from user
submit to create a submit button
date to accept the date from user
email to accept in only email id from user
number to accept a numeric value between a range defined with max and min attribute
range to accept a range between two values specified by max & min attribute
time to accept a time from user
url to accept the url address only
file to accept a multiple media file by user
color to accept the color from user
password to accept a multiple value between a range defined with max, min & maxlength attribute
reset to create a reset button
datetime-local to accept a local date time from user
button to create a button only
month to accept a month from user
search to create a search box on the web page
placeholder set of character to describe a hint about a field
maxlength numeric to set the maxlenght of the character
value set of character to define an initial value for input element
name set of character to define the name of html element
Required it specifies an input filled must be filled with data before submitting the form
Autofocus this attribute focus the input element automatically when user reload the page
Pattern [A-Z, a-z, 1-5]{1,8} this attribute is used to specify the rregular expression on which the input filled value is checked
